## 二、新工具设计方案

### 2.1 代码截图美化工具 (Code Screenshot)

**文件路径**: `tools/dev/code-screenshot.html`

**功能设计**:

| 功能       | 描述                                | 技术实现     |
| ---------- | ----------------------------------- | ------------ |
| 语法高亮   | 支持50+编程语言                     | Prism.js     |
| 主题切换   | 10+预设主题(Dracula/Monokai/Nord等) | CSS变量      |
| 背景设置   | 纯色/渐变/透明                      | CSS渐变      |
| 窗口样式   | macOS/Windows/无边框                | CSS伪元素    |
| 字体选择   | JetBrains Mono/Fira Code/Monaco     | Google Fonts |
| 内边距调整 | 可调节代码周围边距                  | CSS padding  |
| 导出选项   | PNG/SVG/复制到剪贴板                | html2canvas  |
| 行号显示   | 可开关行号                          | Prism插件    |

**UI 草图**:

```
┌─────────────────────────────────────────┐
│ ← 返回                   Code Screenshot │
├─────────────────────────────────────────┤
│ [语言▼] [主题▼] [背景▼] [窗口▼] [字体▼]│
│                                         │
│ ┌─────────────────────────────────────┐ │
│ │ ● ● ●                    untitled   │ │
│ │─────────────────────────────────────│ │
│ │ 1 │ function hello() {              │ │
│ │ 2 │   console.log("Hello World!");  │ │
│ │ 3 │ }                               │ │
│ └─────────────────────────────────────┘ │
│                                         │
│ Padding: [────●────] 64px               │
│                                         │
│ [复制图片] [下载PNG] [下载SVG] [分享]   │
└─────────────────────────────────────────┘
```

**CDN依赖**:

- Prism.js: 语法高亮
- html2canvas: 导出图片

### 2.6 HTTP 请求测试器 (HTTP Client)

**文件路径**: `tools/dev/http-client.html`

**功能设计**:

| 功能     | 描述                              |
| -------- | --------------------------------- |
| HTTP方法 | GET/POST/PUT/DELETE/PATCH/OPTIONS |
| 请求头   | 自定义Headers                     |
| 请求体   | JSON/Form/Raw                     |
| 认证     | Basic Auth / Bearer Token         |
| 响应展示 | 格式化JSON、状态码、耗时          |
| 历史记录 | 本地保存请求历史                  |
| 代码生成 | 生成cURL/fetch/axios代码          |

**注意**: 受CORS限制，需提示用户：

- 仅支持允许跨域的API
- 推荐配合浏览器扩展使用

## 三、技术实现要点

### 3.1 通用技术栈

```javascript
// CDN 依赖列表
const dependencies = {
  // 代码高亮
  prism: 'https://cdn.jsdelivr.net/npm/prismjs@1.29.0/prism.min.js',

  // 图表
  chartjs: 'https://cdn.jsdelivr.net/npm/chart.js@4.4.1/dist/chart.umd.min.js',

  // PDF处理
  'pdf-lib': 'https://cdn.jsdelivr.net/npm/pdf-lib@1.17.1/dist/pdf-lib.min.js',
  pdfjs: 'https://cdn.jsdelivr.net/npm/pdfjs-dist@3.11.174/build/pdf.min.js',

  // 图片导出
  html2canvas: 'https://cdn.jsdelivr.net/npm/html2canvas@1.4.1/dist/html2canvas.min.js',

  // ZIP打包
  jszip: 'https://cdn.jsdelivr.net/npm/jszip@3.10.1/dist/jszip.min.js'
};
```

### 3.2 设计规范

遵循项目现有设计系统:

- 深色主题: `--bg-deep: #0a0a0f`
- 主色调: `--accent-cyan: #00f5d4`
- 字体: JetBrains Mono (代码) / Space Grotesk (UI)

### 3.3 功能标准

每个工具必须实现:

- ✅ 粘贴按钮
- ✅ 复制/下载按钮
- ✅ 分享链接 (URL hash状态)
- ✅ 清空按钮
- ✅ localStorage自动保存

## 六、风险与注意事项

1. **CORS限制**: HTTP测试工具功能受限，需明确提示用户
2. **文件大小**: PDF处理可能遇到大文件性能问题，需设置上限
3. **浏览器兼容**: Canvas导出需测试各浏览器兼容性
4. **版权问题**: 社交媒体模拟工具需添加免责声明
