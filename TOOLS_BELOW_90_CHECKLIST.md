## 优先优化列表 (75-79分)

这14个工具得分最低，应该首先优化。

### 优化方向

对于这个分数段的工具，通常存在以下问题:

1. **可访问性** (最关键)
   - [ ] 缺少H1标签或语义化HTML
   - [ ] ARIA标签不足 (role, aria-label 缺失)
   - [ ] 键盘导航不完整 (Tab键无法访问所有功能)
   - [ ] 焦点样式不可见

2. **色彩对比度** (WCAG AA标准)
   - [ ] 文本对比度 < 4.5:1
   - [ ] 图形/UI元素对比度 < 3:1
   - [ ] 深色主题下可读性差

3. **响应式设计**
   - [ ] 375px (手机) 宽度下布局混乱
   - [ ] 触摸目标 < 44x44px
   - [ ] 存在水平滚动

4. **表单易用性**
   - [ ] 输入框缺少关联标签
   - [ ] 错误提示不清晰
   - [ ] 缺少帮助文字

### 特定工具优化步骤

#### 1. Font Preview (77分)

**问题**:
- 可访问性最薄弱
- 键盘导航有限
- 缺少文字大小切换的ARIA提示

**优化步骤**:
1. 添加主要H1标签
2. 为字体预览区域添加role="region" 和 aria-label
3. 为字体大小控制添加aria-valuemin/max/now
4. 确保Tab键可以到达所有控件
5. 添加:focus-visible样式

**预期效果**: 77分 → 85分

---

#### 2. 烹饪计时器 (77分)

**问题**:
- 移动端触摸目标过小
- 计时器显示的无障碍性不足
- 缺少时间变化的aria-live通知

**优化步骤**:
1. 增加按钮大小到至少48x48px
2. 添加aria-live="polite"到计时器显示区域
3. 为开始/停止按钮添加aria-pressed状态
4. 优化375px宽度的布局
5. 增加字体大小到16px以上

**预期效果**: 77分 → 86分

---

**优化步骤**:
1. 为代码块添加role="region"和aria-label="Curl command"
2. 增加复制按钮大小到48x48px
3. 为错误信息添加role="alert"
4. 优化在小屏幕下的布局
5. 为代码输入/输出添加键盘快捷键

## 优化模板代码

### 基础可访问性改进

```html
<!-- 1. 添加语义化结构 -->
<header role="banner">
  <h1>工具名称</h1>
  <nav aria-label="主导航">
    <a href="#main-content" class="skip-link">跳过导航</a>
  </nav>
</header>

<main id="main-content">
  <!-- 2. 表单无障碍 -->
  <form>
    <label for="input-id">输入标签:</label>
    <input
      id="input-id"
      type="text"
      aria-describedby="help-text"
    />
    <span id="help-text">帮助文字</span>
  </form>

  <!-- 3. 按钮无障碍 -->
  <button
    aria-label="清除输入"
    class="icon-button"
  >
    ✕
  </button>

  <!-- 4. 动态内容更新 -->
  <div aria-live="polite" role="status">
    结果将在这里显示
  </div>
</main>

<style>
  /* 色彩对比度 (WCAG AA) */
  body {
    color: #1a1a1a;      /* 文本 */
    background: #ffffff;  /* 背景 */
  }

  /* 焦点样式 */
  button:focus-visible,
  input:focus-visible,
  a:focus-visible {
    outline: 3px solid #00f5d4;
    outline-offset: 2px;
  }

  /* 触摸目标大小 */
  button,
  input[type="button"],
  input[type="submit"] {
    min-width: 44px;
    min-height: 44px;
    padding: 10px 16px;
  }

  /* 响应式布局 */
  @media (max-width: 375px) {
    button {
      min-width: 48px;  /* 更大的移动端目标 */
    }
  }
</style>
```

### 快速检查清单

对每个工具执行:

```
- [ ] H1标签存在且唯一
- [ ] 主要区域有aria-label或aria-labelledby
- [ ] 表单输入都有关联标签
- [ ] 按钮/链接有清晰的文本或aria-label
- [ ] 颜色对比度 >= WCAG AA (4.5:1文本)
- [ ] 所有功能可通过键盘操作
- [ ] 焦点指示器可见 (:focus-visible)
- [ ] 图片有alt文本 (或aria-hidden="true")
- [ ] 列表结构正确 (<ul>/<ol>/<li>)
- [ ] 没有自动播放的音频/视频
- [ ] 页面语言标记正确 (lang属性)
- [ ] 移动端375px宽度测试通过
```
