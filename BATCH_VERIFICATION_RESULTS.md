## 从业者指南

### 优化模板工具

针对75-89分工具的优化建议:

```html
<!-- 1. 添加语义化标签 -->
<header role="banner">
  <nav role="navigation" aria-label="主导航">
    <a href="#main-content" class="skip-link">跳过导航</a>
  </nav>
</header>

<main id="main-content">
  <h1>工具名称</h1>

  <!-- 2. 表单无障碍 -->
  <form>
    <label for="input-id">输入标签:</label>
    <input
      id="input-id"
      type="text"
      aria-describedby="help-text"
      required
    />
    <span id="help-text">帮助文字</span>
  </form>

  <!-- 3. 按钮无障碍 -->
  <button
    aria-label="清除输入"
    aria-pressed="false"
  >
    清除
  </button>

  <!-- 4. 色彩对比度 -->
  <style>
    /* 确保文本对比度 >= 4.5:1 -->
    body {
      color: #1a1a1a; /* 深色文本 */
      background: #ffffff; /* 亮色背景 */
    }
  </style>
</main>
```

### 验证清单

```
- [ ] H1标签存在且唯一
- [ ] ARIA标签正确使用 (非冗余)
- [ ] 表单所有输入都有关联标签
- [ ] 按钮和链接都有明确文本或aria-label
- [ ] 颜色对比度 >= WCAG AA
- [ ] 键盘可以操作所有功能 (Tab, Enter, Space)
- [ ] 焦点样式可见 (:focus-visible)
- [ ] 图片有alt文本 (或aria-hidden="true")
- [ ] 列表结构正确 (<ul>/<ol>/<li>)
- [ ] 跳过导航链接存在
```
