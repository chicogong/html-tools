💡 资源使用建议

  • 设计灵感： 先去 Dribbble/Behance 找视觉灵感，再去 Mobbin 看实际产品的 UI 模式
  • 组件库选择： React 推荐 Shadcn/ui + Tailwind，Vue 推荐 Naive UI 或 Element Plus
  • 图标库： 首选 Lucide 或 Heroicons，需要更多图标用 Iconify
  • 配色： 先用 Coolors 生成方案，再用 Realtime Colors 预览实际效果
  • 素材授权： 使用前务必确认授权范围，特别是商业项目
