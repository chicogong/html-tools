粘贴访问日志，统计 Top IP、路径、状态码、带宽，纯本地处理

Top N:

0 总请求数
0 唯一 IP
0 总流量
0% 错误率(4xx+5xx)
0 平均响应大小
0 解析成功行
每小时请求量分布
Top IP 地址
# IP 地址 请求数 占比
Top 请求路径
# 路径 请求数 占比
状态码分布
状态码 含义 请求数 占比
Top User-Agent
# User-Agent 请求数 占比
0 请求
0 唯一IP
0 流量
