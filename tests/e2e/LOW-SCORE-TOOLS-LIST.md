## 85分工具的主要问题

**共 178 个工具**，都因 `formUsability = 0` 而卡在 85 分

### 原因
这些工具都有表单输入元素 (<input>, <select>, <textarea>) 但缺乏：
- 关联的 `<label>` 标签
- 或完整的 `aria-label` 属性
- 导致屏幕阅读器无法识别表单字段的用途

### 修复方案
```html
<!-- 修复前：没有标签 -->
<input type='color' value='#667eea' />
<select id='type'></select>

<!-- 修复后：添加标签 -->
<label for='color'>选择颜色</label>
<input type='color' id='color' value='#667eea' />

<label for='type'>选择类型</label>
<select id='type' aria-label='选择类型'></select>
```
