层叠上下文触发条件说明
以下属性会 新建 一个层叠上下文，使子元素的 z-index 相互独立：
· position: relative/absolute/fixed/sticky 且 z-index ≠ auto
· opacity < 1
· transform ≠ none
· isolation: isolate
· will-change 、 filter 、 perspective 等

子元素的 z-index 只在 同一层叠上下文 内比较，跨上下文比较其 父容器 的 z-index。
