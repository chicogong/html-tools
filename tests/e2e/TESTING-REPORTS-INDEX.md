## 测试脚本和验证

### 运行测试

```bash
# 重新测试工具 250-500
node tests/e2e/enhanced-ui-checker.cjs --start=250 --end=500 --output=tests/e2e/verify-batch-250-500.json

# 运行所有 750 工具测试
node tests/e2e/enhanced-ui-checker.cjs --output=tests/e2e/all-tools-audit.json

# 运行代码质量检查
npm run lint
npm run format:check
```

### 验证修复效果

```bash
# 修复后重新运行测试并对比分数
node -e "
  const before = require('./tests/e2e/batch-250-500.json');
  const after = require('./tests/e2e/verify-batch-250-500.json');
  const improved = after.filter(t => {
    const old = before.find(b => b.path === t.path);
    return old && t.score > old.score;
  });
  console.log('Improved tools:', improved.length);
  console.log('New average:', (after.reduce((s,t)=>s+t.score,0)/after.length).toFixed(2));
"
```
